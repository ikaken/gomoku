# 五目並べ (Gomoku)

ブラウザで遊べる五目並べゲームです。

## デモ

🎮 **[プレイする](https://ikaken.github.io/gomoku/)**

## 機能

- **対人戦モード** - 2人で交互に石を置いて対戦
- **CPU対戦モード** - コンピュータと対戦（評価関数ベースのAI）
- **音声読み上げ** - 3連・4連の通知、勝敗結果を音声で読み上げ
- **視覚的フィードバック** - 最後に置いた石のマーク、勝利ラインのハイライト
- **レスポンシブデザイン** - PC・スマートフォン両対応

## 遊び方

1. `index.html` をブラウザで開く
2. 画面上部でゲームモードを選択（対人戦 / CPU対戦）
3. 盤面をクリックして石を置く
4. 先に5つの石を縦・横・斜めに並べた方が勝ち
5. 「リセット」ボタンで新しいゲームを開始

## 技術仕様

- **盤面サイズ**: 15×15
- **使用技術**: HTML5, CSS3, JavaScript (Vanilla)
- **フォント**: Noto Sans JP (Google Fonts)
- **音声合成**: Web Speech API

## CPU AIについて

CPUは以下の優先順位で手を決定します：

1. 自分が5連を作れる手があれば打つ
2. 相手の5連を阻止する
3. 評価関数で最も高いスコアの手を選択
   - 連続数と両端の空き状況を評価
   - 中央に近い位置を優先
